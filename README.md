# enterprise-web-security-assessment
Simulated enterprise web application security assessment with threat modeling, risk analysis, and remediation recommendations.

## Overview
This project simulates a security assessment of a customer-facing web application
used in a large telecom enterprise environment.

## Business Context
The assessed application represents a customer self-service portal that allows
end users to manage personal and billing information, while an internal CSR panel
enables employees to manage customer accounts.

## Objective
The goal of this assessment is to identify security weaknesses that could lead to
account compromise, sensitive data exposure, or regulatory risk.

## Scope
### In Scope
- Web application (frontend and backend)
- Authentication and authorization mechanisms
- Role-based access control
- API endpoints

### Out of Scope
- Denial of Service attacks
- Social engineering
- Physical security

## Methodology
- Threat modeling
- Manual security testing
- Risk-based analysis
- Remediation recommendations
